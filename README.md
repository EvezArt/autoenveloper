# Autoenveloper

Autonomous developer agent — cipher generation, geometric analysis, linguistic patterns, cross-domain structural isomorphism.

## Quick start
```bash
python scripts/test_autoenveloper.py
```

## Modules
- `tools/ciphage_gen.py` — Cipher gen + cross-domain isomorphism detector
- `tools/geometric.py` — Spectral/fractal/symmetry analysis
- `tools/linguistic.py` — N-grams, style fingerprint, PII anonymization
